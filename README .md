# Responsive Landing Page

A clean and fully responsive landing page built using pure HTML and CSS.
This project focuses on layout structuring, flexbox usage, and responsive design.

## 🚀 Features
- Fully responsive (mobile + desktop)
- Flexbox-based layout
- Clean UI structure
- Hover interactions
- Semantic HTML5

## 🛠 Tech Used
- HTML5
- CSS3 (Flexbox)
